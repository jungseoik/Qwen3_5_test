# 오탐 로그 데이터셋 통계

`src/dataset/stats.py`는 카테고리별 / 날짜별 오탐 로그(썸네일, 비디오) 개수를 집계해 표로 출력합니다. 표준 라이브러리만 사용하므로 별도 설치가 필요 없습니다.

## 기대하는 디렉토리 구조

지정한 경로가 아래 계층을 따른다고 가정합니다.

```
{root}/
└── by_category/
    └── {category}/              # 예: falldown, fire, smoke, violence
        ├── thumbnail/
        │   └── {YYYYMMDD}/      # 날짜 폴더
        │       └── *.jpg        # 썸네일 이미지
        └── video/
            └── {YYYYMMDD}/
                └── *.mp4        # 비디오
```

- `by_category/` 가 없으면 오류를 출력하고 종료합니다.
- 같은 날짜에서 썸네일 수와 비디오 수가 다르면 상세 표에 `<-- 불일치` 로 표시합니다.

## 실행

```bash
python src/dataset/stats.py
```

기본 경로는 `nas/data/khonkaen` 입니다. 다른 경로는 `--path` 로 지정합니다.

```bash
python src/dataset/stats.py --path nas/data/khonkaen
python src/dataset/stats.py --path /data/site_a --summary-only
```

## 옵션

| 옵션 | 기본값 | 설명 |
|---|---|---|
| `--path` | `nas/data/khonkaen` | 데이터셋 루트 경로 (`by_category/` 를 포함해야 함) |
| `--summary-only` | off | 카테고리별 날짜 상세 표를 생략하고 전체 요약만 출력 |

## 출력 항목

1. **카테고리별 날짜 상세** — 각 카테고리에 대해 날짜별 썸네일 / 비디오 개수와 소계 (`--summary-only` 시 생략)
2. **전체 요약 (카테고리 중심)** — 카테고리별 합계(썸네일, 비디오, 날짜 수)와 전체 합계
3. **날짜별 요약 (날짜 중심)** — 날짜 × 카테고리 매트릭스. 각 날짜에 카테고리별로 몇 건인지 한눈에 보는 표 (썸네일 기준)

카테고리 중심과 날짜 중심 요약은 분기 없이 **항상 함께** 출력됩니다.

```
  ...

        합계       300      1508       164       688    2660
```
